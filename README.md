#NY times MERN app
This is a fullstack MERN New York Times Search app created with react cli create-react-app and using Node, Express and Axios. 


##to start the app
Fork the repo:
You need to install Mongo, Then start mongo and mongo deamon as below

```
mongod
mongo
```

```
npm install
npm start
```
Goto localhost:3000
